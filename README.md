# Life-Canvas

Sistema de Planificación y Organización Personal.

## Estructura del Proyecto

- **/client**: Frontend (React + Vite). Interfaz de usuario.
- **/server**: Backend (NestJS + PostgreSQL). Lógica de negocio y base de datos.
- **/docs**: Documentación del sistema.

## Cómo Ejecutar

### Opción Rápida

Haz doble clic en el archivo **`LifeCanvas.bat`** en esta carpeta.

### Desarrollo

1. **Backend**:
   ```bash
   cd server
   npm run start:dev
   ```

2. **Frontend**:
   ```bash
   cd client
   npm run dev
   ```

## Requisitos

- Node.js
- PostgreSQL
